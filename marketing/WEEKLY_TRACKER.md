# ISOGuardian — Weekly Marketing Tracker

## How to Use
Run this at the start of each day or week. Claude reads the 90-day calendar, generates today's content, and tracks what's been posted.

At the end of each week (Sunday), generate a weekly report covering:
- What was planned vs. posted
- Any engagement metrics (add manually)
- Adjustments for next week
- Content performance notes
